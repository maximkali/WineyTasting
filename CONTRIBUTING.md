# Contribution Guide

## Getting Started
1. `npm install`
2. `cp .env.example .env`
3. `npm run dev`

## Code Standards
- Pre-commit hooks run:
  - ESLint
  - Prettier
  - Type checking
- 100% test coverage for new features

## PR Checklist
- [ ] Type safety verified
- [ ] Documentation updated
- [ ] Tests added
- [ ] Changelog entry
